카이 X-mailbox 디자인

OVERVIEW

Kai 한국항공우주산업주식회사의 X-mailbox 화면을 디자인하였습니다. 기존에 불편했던 UI 디자인을 개선하여, 직설적인 아이콘 디자인과 포인트 컬러의 통일감을 적용하여 보다 깔끔하고 사용자가 사용하기 편리한 환경을 제공하였습니다. 새로운 디자인에서는 사용자가 필요로 하는 기능들을 직관적으로 파악할 수 있도록 하였으며, 사용자 경험을 향상시키기 위해 레이아웃과 아이콘 등을 재구성하였습니다. 또한, 포인트 컬러를 활용하여 중요한 요소를 강조하고 시각적인 일관성을 유지하였습니다. 이를 통해 Kai 한국항공우주산업주식회사의 X-mailbox 화면은 더욱 직관적이고 효율적인 디자인으로 업그레이드되었으며, 사용자들의 작업 효율을 향상시킬 수 있도록 하였습니다.

업무환경을 개선해줄 X-mailbox UI 디자인

업무환경에서 최대한의 편리함을 제공하는것이 주 목표인 카이 X-mailbox는 기존에 사용하였던 위치와 UI 동일하게 디자인 하여, 기존 사용자가 혼란 없이 사용하도록 하였다.
또한 기존에 텍스트만으로만 되어있어 직설적이지 않았던 메뉴들에 아이콘을 디자인 하여 넣음으로써 직관적으로 메뉴들을 알 수 있도록 사용자의 편의를 고려하였다.
컨텐츠 영역 부분은 최대한 많은 내용을 봐야 한다는 고객사의 요청에 맞게 여백을 최소한으로 디자인하였다.
