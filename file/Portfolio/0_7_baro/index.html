지적측량 바로처리센터사이트

OVERVIEW

기존에 있던 지적측량바로처리센터사이트를 재코딩하였다.
jquery로 박스슬라이드, 배너슬라이드, 패밀리사이트, Gnb, SubGnb, 빠른검색기능을 적용하였다. 기존과 다른 패턴배경을 디자인하여 적용하고, 퀵메뉴의 비주얼, 콜센터 디자인을 변경하였다.
