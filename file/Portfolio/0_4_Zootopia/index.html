주토피아영화홍보 　

OVERVIEW

가상의 디즈니 주토피아 영화 홍보페이지를 디자인을 주제로 잡아 준 토피아만의 컬러와 캐릭터의 조화를 살려 디자인하려고 노력하였다. 팝업 페이지와 캐릭터들의 동적인 효과를 주어 재미를 더하고, 어린아이들이 보는 애니메이션이라는 느낌보다 모든 연령층에게 사랑받는 디즈니 애니이기에 너무 유치함에 치우치지 않고 절제와 강조를 잘 혼합할 수 있도록 노력하였다. 복잡한 프로모션 디자인보다는 중요한 예고편과 등장인물/줄거리의 메뉴로 단축함으로써 애니메이션의 홍보를 더 극대화할 수 있도록 노력하였다. Jquery를 이용하여 팝업 이미지의 효과를 주어 재미를 더하였다.
