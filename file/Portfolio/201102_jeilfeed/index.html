천하제일사료 그룹웨어 리뉴얼 　

OVERVIEW

기존에 있던 천하제일사료 그룹웨어를 리뉴얼하였습니다. 이번 리뉴얼에서는 고객사의 요청에 따라 그룹웨어임에도 불구하고 홈페이지처럼 보이도록 하는 것에 주안점을 두었습니다. 이를 위해 각 페이지의 디자인 요소와 배치를 신중하게 고려하여 사용자가 그룹웨어를 이용하는 동안 편안한 웹 사이트를 경험할 수 있도록 디자인했습니다. 또한, 천하제일사료의 로고 포인트 컬러를 활용하여 그룹웨어의 컬러 스키마를 결정했습니다. 이를 통해 브랜드의 시각적 일관성을 유지하고 사용자들에게 익숙한 느낌을 제공하였습니다. 컨텐츠와 기능의 가시성을 높이기 위해 각 페이지에 적절한 컨텐츠 구성과 시각적 요소를 추가하여 사용자들이 필요한 정보를 쉽게 찾을 수 있도록 하였습니다. 이렇게 리뉴얼된 천하제일사료 그룹웨어는 홈페이지와 같은 느낌을 주면서도 그룹웨어의 기능을 충실히 수행할 수 있는 디자인을 완성하였습니다.
