세계인의축제올림픽 Infographic

OVERVIEW

세계인의 축제인 올림픽을 주제로 정보를 시각화하여 표현한다. 현재까지 열린 올림픽의 위치와 메달을 많이 보유한 국 가 순위, 그리고 세계기록을 세운 선수와 현재까지 열 린 올림픽의 메달 디자인의 변화모습과 개최국의 정보 들을 한국에 볼 수 있다. 표현기법은 모든 사람이 한눈에 쉽게 이해할 수 있도록 픽토그램과 올림픽 공식 아이콘, 일러스트를 사용하여 표현하였다
