TCCSTEEL ERP 시스템 구축

OVERVIEW

TCCSTEEL에서 ERP 포탈 사이트를 구축하기 위해 메인, 서브, 에러 페이지 등의 디자인 작업을 진행했습니다. 이를 위해 스틸의 메인 컬러 중 하나인 블루 계열을 활용하여 일관된 시각적인 통일성을 유지하였습니다. 메인 레이아웃은 고객사의 요구에 따라 메인 배너 이미지 영역과 게시물 2개를 포함하도록 디자인하였습니다. 디자인 작업 외에도 코딩 작업까지 전반적으로 담당하여, 디자인된 레이아웃을 웹 페이지로 구현하였습니다. 이 과정에서 반응형 디자인을 적용하여 다양한 디바이스에서 웹 페이지가 최적으로 보이도록 보장하였습니다. 또한, 서브 페이지와 에러 페이지도 메인 페이지와 시각적으로 일관성 있게 디자인하여 사용자들이 일관된 경험을 얻을 수 있도록 하였습니다. 이를 통해 TCCSTEEL의 ERP 포탈 사이트는 사용자들에게 직관적이고 일관된 인터페이스를 제공하며, 기업의 전반적인 이미지와 브랜드 컬러를 효과적으로 전달할 수 있도록 구축되었습니다.
