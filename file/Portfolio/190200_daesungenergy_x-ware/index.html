대성에너지 그룹웨어 부분 리뉴얼 　

OVERVIEW

대성 에너지의 그룹웨어 PC 로그인 화면, 모바일 로그인 화면, 모바일 메인 화면을 디자인했습니다. 이 프로젝트는 기존 그룹웨어의 리뉴얼 작업으로, 친환경 에너지를 추구하는 대성 에너지의 이미지를 반영하여 친환경 일러스트를 사용해 디자인하였습니다. APP 메인 화면은 아이콘을 이용하여 사용자 편리성을 강조했습니다. PC 로그인 화면은 코딩까지 직접 작업했으며, 모바일 화면은 디자인을 완료했습니다.
