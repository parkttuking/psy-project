KEB하나은행
AML 시스템 디자인 　

OVERVIEW

"KEB하나은행"의 외주직원 관리를 위한 포탈 사이트를 디자인했습니다. 고객사의 요청에 따라 외주직원 관리 현황을 한눈에 볼 수 있도록 아이콘화하였고, 업무 효율성을 높이기 위해 헤더를 왼쪽에 배치했습니다. 기존 레이아웃을 최대한 반영하여, 사이트 변경 후에도 직원들이 쉽게 적응할 수 있도록 했습니다. 디자인은 하나은행의 메인 컬러와 서브 컬러를 참고하여 통일감을 유지했습니다.
