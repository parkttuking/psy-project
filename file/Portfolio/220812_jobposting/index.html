채용공고 디자인

OVERVIEW

기업 내 복지 혜택을 발굴하여 채용 공고에 추가함으로써, 지원자들이 다니고 싶은 회사처럼 보이도록 디자인 작업을 진행하였습니다. 이를 위해 사무실과 카페 테라스의 사진을 직접 촬영하고 편집하여 시각적으로 매력적인 이미지를 제공하였습니다. 또한, 채용 공고 사이트에 HTML로 코딩하여 버튼에 링크를 추가함으로써 사용자들이 쉽게 관련 정보를 확인할 수 있도록 하였습니다. 이 작업을 통해, 회사의 복지 혜택을 효과적으로 홍보하고, 지원자들에게 긍정적인 인상을 심어줄 수 있도록 하였습니다.
