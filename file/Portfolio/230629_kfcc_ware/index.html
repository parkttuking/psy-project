OVERVIEW

"MG새마을금고"의 그룹웨어 리뉴얼 디자인을 진행하였습니다.고객사의 요청에 따라 기존 레이아웃을 유지하는 형태로 디자인하였으며, 레이아웃은 변경하지 않는 대신 사용자 편의성을 높이는 데 중점을 두었습니다. 기존 그룹웨어의 불편함을 개선하고자, 자주 사용하는 메뉴에 쉽게 접근할 수 있도록 퀵메뉴를 추가하였습니다. 또한, 업무 체크 사항을 한눈에 확인할 수 있도록 메인 포틀릿에 배치하여 사용자들이 필요한 정보를 빠르게 파악할 수 있도록 설계하였습니다. 이를 통해 사용자 경험을 크게 향상시켰습니다.
