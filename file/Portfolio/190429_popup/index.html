TCCSTEEL 팝업창디자인

OVERVIEW

TCCSTEEL에서 슬로건 공모전을 임직원에게 알리는 팝업창을 디자인했습니다. 이 디자인의 주요 컨셉은 일러스트를 활용하여 부드럽고 친근한 느낌을 전달하는 것입니다. 디자인에는 부드러운 톤의 색상과 친근한 일러스트가 사용되어 직원들에게 다가갈 수 있도록 조화롭게 조합되었습니다. 이를 통해 슬로건 공모전에 대한 관심을 높이고 참여를 유도하도록 하였습니다.
