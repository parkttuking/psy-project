사내 WEB CHAT
디자인 & 코딩 　

OVERVIEW

사내 그룹웨어와 메신저 앱의 로그인 페이지, 목록 페이지, 채팅 뷰 페이지를 디자인하였습니다. 회사의 메인 컬러로 블루 계열을 사용하여 통일성을 부여하였습니다. 업무에 활용되는 메신저이므로, 본연의 기능에 영향을 주지 않으면서도 심플하고 직관적인 디자인을 적용하였습니다. 로그인 페이지는 사용자가 쉽게 로그인할 수 있도록 단순하면서도 명확한 인터페이스를 제공하였습니다. 목록 페이지는 사용자가 중요한 정보를 빠르게 확인할 수 있도록 구성하였으며, 필요한 기능에 빠르게 접근할 수 있도록 배치하였습니다. 채팅 뷰 페이지는 메시지의 가독성을 고려하여 깔끔한 디자인으로 제공되었습니다. 이러한 디자인은 사용자들이 업무를 보다 효율적으로 수행할 수 있도록 도와주며, 편리한 사용자 경험을 제공합니다.
