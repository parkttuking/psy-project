밀크오름 관리자페이지

OVERVIEW

천하제일사료의 밀크오름 관리자 페이지를 디자인하였습니다. 고객사의 요청에 따라 콘텐츠가 많지 않다는 점을 고려하여, 스크롤이 필요 없는 형태로 디자인하였습니다. 이로 인해 관리자들이 필요한 정보를 한 화면에서 모두 확인할 수 있도록 구성하였으며, 직관적이고 효율적인 UI/UX를 구현하였습니다. 깔끔하고 심플한 레이아웃을 적용하여 사용자가 쉽게 이해하고 사용할 수 있도록 하였으며, 주요 기능과 정보를 한눈에 파악할 수 있게 배치하여 관리자 업무의 편의성을 극대화하였습니다.
