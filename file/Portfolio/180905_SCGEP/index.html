OVERVIEW

서울도시가스 SCG EP 모바일 웹은 서울도시가스 임직원들을 위한 그룹웨어 APP 서비스입니다. 이 서비스는 임직원들의 업무 효율을 높이기 위해 설계되었습니다. UI 디자인은 심플함과 브랜드 컨셉을 중시하여 깔끔하고 직관적으로 구성하였습니다. 또한, 해야 할 일의 개수를 명확하게 표시하여 임직원들이 업무를 놓치지 않고 신속하게 처리할 수 있도록 도와줍니다. 이러한 디자인 요소들을 통해 사용자 경험을 향상시키고, 업무 효율성을 극대화할 수 있도록 하였습니다.
