아뜰리에 뷰티아카데미

OVERVIEW

아뜰리에 뷰티아카데미는 중고등학생을 대생으로 미용대에 입학할 목적으로 입시를 준비할수있게 도와주는 학원입니다. 아뜰리 모바일사이트를 많이 접속할 연령대인 10~20 초반을 타겟으로 한 디자인과 아이콘으로 타겟 연령대에 맞는 컬러와 디자인을 표현하려 노력하였고 퀵아이콘과 한눈에 보이는 배너링크를 통하여 손쉽게 접근할 수 있게하였다.
